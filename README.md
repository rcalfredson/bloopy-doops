# bloopy-doops (ブロッピー デュパー)
Mini-scripts to expedite drudgery of managing files, metadata, et cetera.

ファイル、メタデータなどの管理の手間を軽減するミニスクリプト。

## Prerequisites (前提条件)
- Ruby
- Bundler

## Setup (セットアップ)
```
gem install bundler
bundle install

bundle exec ruby _SCRIPT_.rb
```

## Scripts
 - `mu_meta.rb`: generate JSON file of music metadata

## Testing
  - Lint: `rubocop`
